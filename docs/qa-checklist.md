## Screenshot naming

When adding or updating page screenshots, name files using the pattern `<page>-<platform>.png`.

Supported platforms:
- `chrome`
- `edge`
- `mobile`

For example, screenshots for the login page should be saved as:

- `login-chrome.png`
- `login-edge.png`
- `login-mobile.png`
