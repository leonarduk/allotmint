# External Assets

The following binary assets were removed from the repository. To obtain them, use the data URLs stored in this directory.

- **Default Avatar**: see [`default-avatar.url`](default-avatar.url) or the base64 data in [`default-avatar.base64`](default-avatar.base64)
- **Inter Subset Font**: see [`inter-subset.woff2.url`](inter-subset.woff2.url) or the base64 data in [`inter-subset.woff2.base64`](inter-subset.woff2.base64)
- **DejaVu Sans Font**: see [`DejaVuSans.ttf.url`](DejaVuSans.ttf.url) or the base64 data in [`DejaVuSans.ttf.base64`](DejaVuSans.ttf.base64)
- **Menu Screenshot**: see [`menu_1.png.url`](menu_1.png.url) or the base64 data in [`menu_1.png.base64`](menu_1.png.base64)

You can download each asset by pasting the data URL into your browser's address bar or by decoding the corresponding base64 file, for example:

```sh
base64 -d default-avatar.base64 > default-avatar.png
base64 -d inter-subset.woff2.base64 > inter-subset.woff2
base64 -d DejaVuSans.ttf.base64 > DejaVuSans.ttf
base64 -d menu_1.png.base64 > menu_1.png
```
